# Thanglish Transliteration Automation using Playwright

This project contains automated test cases for the Thanglish (Tamil-English) Transliteration website using Playwright.

The automation covers:
- Positive Functional Test Cases
- Negative Functional Test Cases
- UI Test Cases

---

## Technologies Used
- Node.js
- Playwright
- JavaScript
- VS Code

---

## Project Structure

tests/
 ├ positive.spec.js   - 24 Positive Functional Test Cases  
 ├ negative.spec.js   - 10 Negative Functional Test Cases  
 └ ui.spec.js         - 2 UI Test Cases  

---

## Prerequisites
- Install Node.js
- Install VS Code

---

## Installation Steps

1. Open the project folder in VS Code  
2. Open terminal  
3. Run the following commands:

npm install  
npx playwright install  

---

## How to Run Tests

Run this command in terminal:

npx playwright test
